# gu-jedi
I wanna be a Jedi Knight

# Development Flow
リポジトリをクローンしてきます（ビルドに必要なファイルが落ちてきます）．

```bash
$ git clone https://github.com/y-sira/gu-jedi.git
```

ブランチを切って，

```bash
$ git branch -b {BRANCH_NAME}
```

作業します．

作業が完了したら変更したファイルを登録します．

```bash
$ git add {FILE_NAME}
```

コミット時に変更点を書いて，

```bash
$ git commit
```

プッシュします．

```bash
$ git push
```
